# 快速开始指南

<div align="center">

**[简体中文](QUICKSTART.zh-CN.md)** | **[English](QUICKSTART.md)**

</div>

欢迎使用 Focust！本指南将帮助您在几分钟内开始使用您的第一个休息提醒。

## 首次启动

当您首次启动 Focust 时，您将可以在托盘图标中找到它。应用程序**已经在后台运行**，并将根据默认休息计划开始提醒您休息。

### 了解默认设置

开箱即用，Focust 配置为：
- **短休息**：每 20 分钟一次（持续 20 秒）
- **长休息**：每 4 次短休息后（持续 5 分钟）
- **活动时间**：全周全天
- **语言**：从您的系统自动检测

## 基本配置

### 1. 调整休息间隔

如果您想更改休息发生的频率：

1. 转到**休息计划**选项卡
2. 找到默认计划 Default Schedule 部分
3. 在**短休息**下：
   - 将「间隔」更改为您喜欢的时间（例如，30 分钟）
   - 如果您想要更长/更短的休息，请调整「时长」
4. 在**长休息**下：
   - 更改「累计短休次数」（例如，3 而不是 4）
   - 调整「时长」（例如，10 分钟而不是 5 分钟）
5. 点击顶部的**保存更改**

### 2. 自定义休息外观

使休息更具视觉吸引力：

1. 留在**休息计划**选项卡
2. 向下滚动到**主题**部分（在短休息或长休息下）
3. 选择背景：
   - 点击**纯色**并选择一个颜色
   - 或点击**单张图片**并浏览背景图片
   - 或点击**图片文件夹**以选择图片文件夹以获得随机背景
4. 调整文本颜色、模糊和不透明度滑块
5. 点击**保存更改**

### 3. 添加休息声音

要获得音频通知：

1. 在**休息计划**选项卡中，找到**提示音**部分
2. 选择「内置声音」并从以下选项中选择：
   - Gentle Bell - 温和的铃声
   - Soft Gong - 柔和的锣声
   - Notification - 简单的通知
   - Bright Notification - 明亮的通知
3. 调整音量滑块
4. 点击「预览」测试声音
5. 点击**保存更改**

### 4. 设置定时提醒（可选）

对于基于特定时间的提醒（如喝水）：

1. 转到**定时提醒**选项卡
2. 点击**新增提醒**
3. 填写：
   - 名称：「喝水提醒」
   - 标题：「补水时间」
   - 内容：「喝一杯水！」
   - 提醒时间：添加时间，如「10:00」「14:00」「16:00」
4. 选择应该出现的日期
5. 点击**保存更改**

## 使用 Focust

### 休息期间

当休息出现时：
- **计时器**显示剩余时间
- **建议**给您想法（伸展、眼部运动等）
- **恢复按钮**：提前结束休息
- **推迟按钮**：将休息延迟 5 分钟（可配置）
- **键盘快捷键**：按 Enter 完成，或按您配置的键推迟

### 系统托盘

Focust 位于您的系统托盘（通知区域）中：
- **左键单击**：显示设置窗口
- **右键单击**：带有选项的菜单：
  - 显示设置
  - 暂停/恢复
  - 退出

### 暂停休息

有时您需要暂停所有休息（重要会议、演示等）：

1. 点击设置顶部的**暂停**按钮
2. 或右键单击托盘图标 → 暂停
3. 准备好继续时点击**恢复**

### 推迟休息

如果休息在不方便的时候打断：
1. 点击休息窗口中的**推迟**按钮
2. 或按您配置的推迟快捷键（在通用设置中设置）
3. 休息将在 5 分钟（或您配置的推迟持续时间）后重新出现

## 初学者提示

### 1. 从较长的间隔开始
如果 20 分钟的间隔感觉太频繁，请在适应时增加到 30 或 45 分钟。

### 2. 启用通知
打开「休息前通知」（5-10 秒警告），这样休息就不会让您感到惊讶。

### 3. 谨慎使用严格模式
严格模式可防止跳过休息。只有在您真的需要纪律时才启用此功能！

### 4. 为您的休息计划自定义
为以下创建不同的休息计划：
- 工作时间（频繁休息）
- 晚上（较少频繁）
- 周末（可选，或不同的时间）

### 5. 尝试主题
尝试不同的背景图片来放松您。自然场景、简约图案或纯色镇定色效果很好。

## 常见问题

### 「如何在未到休息/提醒时间时手动启动休息/提醒窗口以测试？」
- 如果是为了测试一下主题的效果，「休息计划」设置中就有简单的预览
- 如果想要检查真实的休息/提醒窗口，抑或是背景选择了「图片文件夹」时想看随机效果，可以：
- 转到**高级选项**选项卡
- 点击「高级选项」标题左边的图标，会显示 Debug 调试区域，可以手动启用短休息、长休息、提醒窗口或跳过当前休息（v0.2.3+）

### 「无法通过设置界面/托盘图标将调度器从暂停状态恢复」
- 这是因为调度器是被某个监视器暂停的（例如 DND 监视器），而不是用户手动暂停的
- 手动恢复只能在所有监视器都没有发送暂停命令时才能工作，即手动恢复只能恢复手动暂停导致的暂停状态

### 「休息打断了重要工作！」
- 最初自由使用**推迟**功能
- 设置**全局延后快捷键**以快速延后
- 考虑增加您的休息间隔
- 使用**系统空闲检测**（在通用下）在您离开时自动暂停

### 「我想让休息出现在我所有的显示器上」
- 转到**通用设置**选项卡
- 启用「在所有显示器上显示休息界面」
- 点击**保存更改**

### 「如何使休息全屏？」
- 转到**通用设置**选项卡
- 将「休息窗口大小」设置为 100%
- 休息现在将填满整个屏幕

### 「我可以禁用建议吗？」
- 转到**休息计划**选项卡
- 在您的休息计划的**建议**部分下
- 取消选中「休息时显示建议」
- 点击**保存更改**

### 「我想使用我自己的音频文件」
- 转到**休息计划** → **提示音**
- 选择「自定义文件」
- 点击**浏览**并选择您的 MP3/WAV/OGG 文件
- 调整音量并保存

### 「Windows 下调度器状态频繁暂停」
- 可能是因为启用了「勿扰模式检测」选项，Windows 默认会在一些场景自动启用「请勿打扰」
- 可以 `Win` + `I` 打开设置，导航到「系统」→「通知」，自行配置「请勿打扰」的自动规则
- 或者在 Focust 中禁用该检测选项
